# VERSION HISTORY - PDF 여백 편집기

모든 날짜 및 시간은 아시아/서울(KST) 기준입니다.

## [1.7] - 2026-02-21
### 수정된 기능
- **저장 에러 수정**: "CropBox not in MediaBox" 오류 해결 (단계별 경계 박스 설정 로직 적용)
- **버전 동기화**: 코드와 문서의 버전 번호를 1.7로 일치시킴

## [1.2] - 2026-02-21
### 수정된 기능
- **비압축 저장 회전 보정**: 일부 PDF에서 비압축 저장 시 180도 뒤집히는 현상을 해결하기 위해 회전 로직 수정
- **파일명 변경**: `pdf editor 1.1.py` -> `pdf editor 1.2.py`

## [1.1] - 2026-02-21
### 수정된 기능
- **회전 페이지 저장 버그 수정**: 회전(Rotation) 속성이 있는 PDF 페이지 저장 시 방향 뒤틀림 문제 해결
- **압축 알고리즘 개선**: DPI 고정(200) 및 JPEG 품질 조절 방식으로 화질 저하 최소화

## [1.0] - 2026-02-19
### 추가된 기능
- **PDF 여백 조정**: 홀수/짝수 페이지별 개별 여백 설정 기능
- **실시간 미리보기**: 설정값 실시간 확인 기능
- **단일 실행 파일**: PyInstaller를 통한 .exe 추출
