# VERSION HISTORY - PDF 여백 편집기

모든 날짜 및 시간은 아시아/서울(KST) 기준입니다.

## [1.0] - 2026-02-19
### 추가된 기능
- **PDF 여백 조정**: 홀수/짝수 페이지별 개별 여백 설정 기능 (mm 단위)
- **자르기 및 늘리기**: 양수(+) 입력 시 여백 추가, 음수(-) 입력 시 여백 자르기 지원
- **강력한 압축 기능**: 이미지 리샘플링 기반의 용량 축소 기능 (회전 문제 완벽 해결)
- **실시간 미리보기**: 설정값에 따른 가이드라인(빨간 점선) 및 결과물 실시간 확인
- **화면 제어**: Ctrl + 마우스 휠 및 버튼을 통한 화면 확대/축소 지원
- **프리셋 시스템**: 자주 사용하는 여백 설정을 저장하고 불러오는 기능
- **자동 저장**: 프로그램 종료 시 마지막 설정값을 자동으로 기억
- **진행률 표시**: PDF 변환 및 저장 시 진행 상태바 표시
- **단일 실행 파일**: PyInstaller를 통한 단일 `.exe` 추출 완료
