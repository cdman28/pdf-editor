# PDF 여백 편집기 (PDF Margin Editor) 1.8

스캔된 PDF 문서의 홀수/짝수 페이지 여백을 정교하게 조정할 수 있는 데스크탑 애플리케이션입니다. 특히 제본된 책을 스캔했을 때 발생하는 여백 불균형을 해결하는 데 최적화되어 있습니다.

## 🚀 주요 기능

- **홀수/짝수 페이지 개별 설정**: 제본 위치에 따른 페이지별 맞춤 여백 조정
- **정교한 수치 입력**: mm 단위로 상, 하, 좌, 우 여백을 세밀하게 조절
- **강력한 압축 및 회전 보정**: 이미지 리샘플링 기술을 적용하여 용량을 획기적으로 줄이고, 문서 방향(세로/가로) 문제를 완벽히 해결
- **실시간 미리보기**: 변경사항을 즉시 확인 가능한 인터페이스 및 줌(Zoom) 기능
- **프리셋 기능**: 자주 사용하는 설정을 저장하여 반복 작업 효율화
- **최신 설정 기억**: 재시작 시 마지막 작업 수치를 자동으로 로드

## 🛠 설치 및 실행

### 실행 파일 사용
1. `dist` 폴더 내의 `pdf editor 1.8.exe` 파일을 실행합니다.
2. 추가 설치 과정 없이 바로 사용 가능합니다.

### 소스 코드 실행 (개발자용)
- Python 3.13+ 
- 필수 라이브러리: `PyQt6`, `PyMuPDF (fitz)`
```bash
pip install PyQt6 PyMuPDF
python "pdf editor 1.8.py"
```

## 📖 사용 방법

1. **파일 열기**: [파일 열기] 버튼을 눌러 대상 PDF를 로드합니다.
2. **여백 조절**: 우측 패널의 홀수/짝수 탭을 이동하며 상/하/좌/우 수치를 입력합니다.
   - **양수(+)**: 종이 바깥으로 여백을 추가 (Padding)
   - **음수(-)**: 안쪽으로 여백을 잘라냄 (Crop)
3. **미리보기**: 빨간색 점선(원본 위치)과 흰색 배경(최종 결과)을 확인합니다.
4. **저장 옵션**: 용량 다이어트가 필요한 경우 압축 수준을 조절합니다. (10% 이상 설정 시 회전 자동 보정 적용)
5. **저장**: [저장 하기] 버튼을 눌러 결과물을 생성합니다.
