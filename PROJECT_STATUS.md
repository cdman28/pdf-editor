# 프로젝트 상태 보고서 (Project Status Report)

## 프로젝트 정보
- **프로젝트명**: PDF 여백 편집기 (PDF Margin Editor)
- **위치**: `d:\APP\PDF 편집`
- **주요 기능**: 
  - PDF 파일의 홀수/짝수 페이지별 여백 설정 (자르기 및 늘리기)
  - 실시간 미리보기 (Zoom 지원)
  - 직관적인 GUI (PyQt6 기반)

## 현재 버전: V1.0
- **생성일**: 2026-02-19
- **개발 환경**: Python 3.12, PyQt6, PyMuPDF

## 주요 파일 구성
1. `pdf_editor_v1.py`: 메인 애플리케이션 소스 코드
   - GUI 구현 (좌측 미리보기, 우측 설정 패널)
   - PDF 처리 로직 (fitz 라이브러리 활용)
   - 설정값 관리 (홀수/짝수 분리)

## 사용 방법
1. 터미널에서 `python pdf_editor_v1.py` 실행
2. [PDF 파일 열기] 버튼으로 파일 로드
3. 홀수/짝수 탭을 선택하여 상하좌우 여백 입력
   - 양수(+): 안쪽으로 자르기 (Crop)
   - 음수(-): 바깥쪽으로 여백 추가 (Padding)
   - 단위: mm
4. 미리보기로 빨간색 가이드라인 확인
5. [PDF 저장] 버튼으로 결과물 저장
